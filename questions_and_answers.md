###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: `"12"`

<i>a number (1) and a string ("2") in JavaScript, it converts the number to a string and then combines them. So, the result is 12, not the mathematical sum of 1 + 2. so the answer is C: 12.</i>

</p>
